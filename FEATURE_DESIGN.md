# 小蓝鸟 - 功能扩展设计

> 项目：Tauri 2 + Rust 后端 + 原生 JS 前端
> 已有模块：活动监测、AI日记、心情追踪、习惯打卡、复盘生成、写作计时器、灵感速记、桌面宠物、成就系统

---

## P0 - 核心差异化

### 1. 桌面世界观成长
把用户行为映射成一张可视化小地图/小世界。学习→图书馆区升级，写作→剧场解锁，运动→公园繁荣，熬夜→乌云/怪物出现。每天/每周推进"城市建设"，跟现有活动监测数据打通。

### 2. 小蓝鸟人格系统
给小蓝鸟做性格参数（毒舌/温柔/中二/社恐等滑杆），它的提醒文案、评价、吐槽随人设变化。积累"共同梗库"——用户反复出现的行为模式会变成小蓝鸟的专属梗。

---

## P1 - 强特色功能

### 3. 记忆小剧场
基于当天活动数据和日记，自动生成轻小说/VN风格的"一日番外"——有场景、事件、情绪旁白，甚至给出2-3个"如果当时选了另一条路"的分支选项。

### 4. 今日反派系统
识别用户最容易沉迷的分心源（基于活动监测数据），拟人化成反派角色。用户可以"谈判/封印/放纵一次"，封印是交换条件式："刷10分钟可以，但先写200字"。

### 5. 伏笔管理器
面向网文写作场景。随手记"伏笔卡"（人物、设定、未填坑），检测到写作窗口时弹出相关伏笔提醒，章节结束时问"这个坑要不要今天埋一点"。

---

## P2 - 锦上添花

### 6. 英语词汇拾取
检测到用户在英文网页/文档停留较久时，抓取高频词组，生成风格化例句（现实场景、网文对白、程序员吐槽三种风格），形成"今日拾取道具"。

### 7. 灵感炼金术
对灵感速记做自动聚类，定期给出"合成配方"：A灵感+B情绪+C场景→一个可用的段落或设定，一键收录进世界观/人物卡。

### 8. 小蓝鸟收藏馆
长期数据变成可逛的"博物馆"界面——常用工具展柜、灵感标本、年度关键词云，能导出为海报长图。

### 9. 社交能量条
基于聊天/群聊/会议软件时长生成"社交能量消耗"，过载时小蓝鸟提示切换低社交模式。

---

## 实施顺序建议

**第一批（快速出效果）：**
- 2. 人格系统 ← 改动最小，体验提升最明显
- 9. 社交能量条 ← 纯复用现有数据
- 4. 反派系统 ← 趣味性强，实现简单

**第二批（独立模块）：**
- 3. 记忆小剧场 ← AI调用现成，主要是前端
- 5. 伏笔管理器 ← 独立CRUD，不影响其他模块
- 6. 英语词汇拾取 ← 可以先做简化版

**第三批（大工程）：**
- 7. 灵感炼金术 ← 等灵感速记数据积累够了再做
- 1. 桌面世界观 ← 最重，建议单独立项
- 8. 收藏馆 ← 等数据量足够再做才有意义
